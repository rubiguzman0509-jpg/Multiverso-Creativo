Portafolio Web Personal – Ruby Esther Guzmán

Descripción

Este proyecto corresponde a mi portafolio web personal, desarrollado como parte de la asignatura de Programación Web. En este sitio presento una selección de mis trabajos académicos y proyectos creativos, integrando mis habilidades como diseñadora gráfica, fotógrafa y desarrolladora web en formación.

El objetivo del portafolio es mostrar de manera organizada, visualmente atractiva y funcional mi evolución dentro del área multimedia, aplicando principios de diseño, experiencia de usuario y desarrollo frontend.



Enfoque Profesional

Como diseñadora, me interesa combinar lo visual con lo técnico, creando experiencias digitales modernas que sean tanto estéticas como funcionales. Este portafolio refleja mi enfoque hacia el diseño centrado en la usuaria, la adaptabilidad y la comunicación visual efectiva.



Tecnologías Utilizadas

* HTML5 (estructura semántica)
* CSS3 (diseño visual, Flexbox y Grid)
* JavaScript (interactividad básica)
